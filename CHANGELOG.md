# Changelog

All notable changes to countwise are documented here.

---

## [v1.0] — 2026-03-12

Initial release.

### Fabric Calculator
- Calculate cut fabric size across all common Aida and evenweave counts
- Metric and imperial unit toggle
- Needle size suggestions per count

### Stitch Stash
- Full DMC thread inventory (500+ colours)
- Track quantity and low-stock threshold per colour
- Organise threads by project
- My Stash, Browse All, and Projects tabs
- Search across all tabs
- Colour family filter pills (consistent across tabs)
- Colour swatches with DMC number tooltip on hover
- Rename and delete projects

### App
- countwise logo in nav and landing page
- Nav and footer on all pages (Fabric Calculator and Stitch Stash)
- Low Noise Labs footer with developer avatar and LNL logo
- Social preview OG image
- Fixed: home page title showing "countwise | countwise"
- Fixed: footer tagline updated to "Small tools. Built carefully."
